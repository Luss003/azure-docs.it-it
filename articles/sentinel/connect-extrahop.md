# <a name="connect-extrahop-revealx"></a>Connetti ExtraHop Reveal (x)

Il connettore ExtraHop Reveal (x) Data consente di connettere facilmente il sistema Reveal (x) con Azure Sentinel per visualizzare i dashboard, creare avvisi personalizzati e migliorare l'analisi. Questa integrazione offre la possibilità di ottenere informazioni dettagliate sulla rete dell'organizzazione e di migliorare le funzionalità di sicurezza.


## <a name="prerequisites"></a>Prerequisiti

- Autorizzazioni di lettura e scrittura per l'area di lavoro di Azure Sentinel.

- ExtraHop individuare o usare l'appliance con firmware versione 7,8 o successiva con un account utente con privilegi illimitati (amministratore).

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>Connettere Azure Sentinel a ExtraHop Reveal (x)

1. Nel portale di Azure passare ad **Azure Sentinel** > **connettori dati** , quindi selezionare il connettore **ExtraHop Reveal (x)** .

2. Selezionare **Apri connettore pagina**.

3. Seguire le istruzioni riportate nella pagina **ExtraHop Reveal (x)** .
