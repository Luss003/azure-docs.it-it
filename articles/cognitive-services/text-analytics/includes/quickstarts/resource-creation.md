Iniziare a usare la API Analisi del testo con una delle opzioni seguenti.


* <a href="https://azure.microsoft.com/try/cognitive-services/my-apis/?api=text-analytics" title="Creare una risorsa di valutazione di Analisi del testo" target="_blank">Creare una risorsa di valutazione <span class="docon docon-navigate-external x-hidden-focus"></span></a>
    * non richiede una sottoscrizione di Azure, 
    * è gratuita e valida sette giorni. Dopo aver eseguito l'iscrizione, una chiave di valutazione e un endpoint saranno disponibili sul [sito Web di Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/). 
    * Si tratta di un'ottima soluzione se si vuole provare l'API Analisi del testo, ma non si ha una sottoscrizione di Azure.
    
* <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesTextAnalytics"  title="Creare una risorsa di Analisi del testo"  target="_blank">Creare una risorsa di Analisi del testo<span class="docon docon-navigate-external x-hidden-focus"></span></a>
    * disponibile tramite il portale di Azure fino a quando la risorsa non viene eliminata.
    * Usare il piano tariffario gratuito per provare il servizio ed eseguire in un secondo momento l'aggiornamento a un livello a pagamento per la produzione.
     
* <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne"  title="Creare una risorsa per più servizi" target="_blank">Creare una risorsa per più servizi<span class="docon docon-navigate-external x-hidden-focus"></span></a>
    * disponibile tramite il portale di Azure fino a quando la risorsa non viene eliminata.  
    * Usare la stessa chiave e lo stesso endpoint per le applicazioni in più servizi cognitivi.
