Ottenere una chiave e un endpoint per autenticare le applicazioni. Creare una risorsa per Analisi del testo usando il [portale di Azure](../../../cognitive-services-apis-create-account.md) o l'[interfaccia della riga di comando di Azure](../../../cognitive-services-apis-create-account-cli.md) nel computer locale. È anche possibile:

* Ottenere un [codice della versione di valutazione gratuita](https://azure.microsoft.com/try/cognitive-services/#decision), valido per 7 giorni. Dopo aver eseguito l'iscrizione, sarà disponibile sul [sito Web di Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/).  
* Visualizzare questa risorsa nel [portale di Azure](https://portal.azure.com/)

Dopo aver ottenuto una chiave e l'endpoint dalla sottoscrizione di valutazione o dalla risorsa, creare due [variabili di ambiente](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account#configure-an-environment-variable-for-authentication). Denominarne una `TEXT_ANALYTICS_SUBSCRIPTION_KEY` per la chiave e una `TEXT_ANALYTICS_ENDPOINT` per l'endpoint.
