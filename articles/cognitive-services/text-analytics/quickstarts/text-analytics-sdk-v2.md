# <a name="quickstart-use-the-text-analytics-client-library-v2"></a>Avvio rapido: Usare la libreria client di Analisi del testo v2

Introduzione alla libreria client di Analisi del testo v2. Seguire questi passaggi per installare il pacchetto e provare il codice di esempio per le attività di base.

* Analisi del sentiment
* Rilevamento della lingua
