## <a name="reference-docs"></a>Documenti di riferimento

Il servizio di riconoscimento vocale fornisce due SDK. Il primo SDK è l' [SDK di sintesi vocale](../speech-sdk.md) principale e fornisce la maggior parte delle funzionalità necessarie per interagire con il servizio di riconoscimento vocale. Il secondo SDK è specifico per i dispositivi, denominati in modo appropriato SDK per i [dispositivi vocali](../speech-devices-sdk.md). Entrambi gli SDK sono disponibili in molte lingue.

### <a name="speech-devices-sdk-reference-docs"></a>Documentazione di riferimento per i dispositivi vocali SDK

[Speech Devices SDK](../speech-devices-sdk.md) è un superset dell'SDK di riconoscimento vocale, con funzionalità estese per dispositivi specifici. Per scaricare l'SDK per i dispositivi vocali, è necessario innanzitutto [scegliere un kit di sviluppo](../get-speech-devices-sdk.md#choose-a-development-kit).
