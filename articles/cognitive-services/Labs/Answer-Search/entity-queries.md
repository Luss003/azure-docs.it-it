# <a name="quickstart-query-for-entities"></a>Guida introduttiva: Query per le entità

Se la query richiede informazioni su una persona, un luogo o una cosa, la risposta può contenere un riscontro `entities`.  Le query restituiscono sempre pagine Web e i [fatti](fact-queries.md) e/o le [entità](entity-queries.md) dipendono dalla query.

Le entità supportano tre scenari di query: 
-   DominantEntity - Esiste una sola entità che corrisponde alla query e alla finalità dell'utente. Ad esempio, la query Space Needle è uno scenario DominantEntity. 
-   Disambiguation - Esiste più di un'entità che corrisponde alla query e alla finalità dell'utente e spetta all'utente selezionare l'entità corretta. Ad esempio, la query Il trono di spade è uno scenario Disambiguation che restituisce sia la serie TV che la serie di libri. 
-   List - Esistono più entità che corrispondono alla query e alla finalità dell'utente. Ad esempio, la query "Elenco delle specie in pericolo" è uno scenario List che restituisce valori tabulari formattati per la visualizzazione in righe e celle. 
 
Per determinare lo scenario di query, usare il campo `queryScenario` dell'oggetto `entities`. I dati che include l'entità dipendono dal tipo dell'entità. Anche se le entità includono le stesse informazioni di base, alcune entità, ad esempio le attrazioni turistiche o i libri, includono proprietà aggiuntive. Le entità che includono proprietà aggiuntive includono il campo `_type` che contiene un hint usato dal serializzatore. Le entità seguenti includono proprietà aggiuntive: 
-   Book 
-   MusicRecording 
-   Person 
-   Attraction 
 
Per determinare il tipo di entità che contiene la risposta, usare il campo `entityTypeHints` come illustrato nella query per Bill Gates.
```
        },
        "description": "Bill Gates is an American business man and philanthropist, co-founder of Microsoft",
        "entityPresentationInfo": {
          "entityScenario": "DominantEntity",
          "entityTypeHints": [
            "Person"
          ]
        },
        "bingId": "6d7d66a7-2cb8-0ae9-637c-f81fd749dc9a"
      }
```

Una query può restituire un elenco se è pertinente.

```
https://api.labs.cognitive.microsoft.com/answerSearch/v7.0/search?q=list+of+endangered+species
```
