# <a name="authoring-api-changes"></a>Modifiche all'API di creazione 

Le API di creazione vengono aggiornate, per V3, per consentire la [scomposizione dei modelli con le entità apprese dal computer](luis-concept-data-extraction.md). 

È possibile creare nelle API v2 o Preview V3. La stessa app può anche scegliere di usare le risposte dell'endpoint di stima v2 o V3. Non è necessario avere la stessa versione dell'API per la creazione e la risposta dell'endpoint di stima.

Il portale LUIS USA esclusivamente le API di creazione di V3.

## <a name="next-steps"></a>Passaggi successivi

Esaminare le [modifiche apportate all'endpoint di stima per l'](luis-migration-api-v3.md) API V3.
