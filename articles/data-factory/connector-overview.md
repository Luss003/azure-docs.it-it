# <a name="azure-data-factory-connector-overview"></a>Panoramica di Azure Data Factory Connector

Azure Data Factory supportano gli archivi dati seguenti tramite l'attività di copia, il flusso di dati, l'attività di ricerca, l'attività Get Metadata e l'attività Delete. Fare clic su ogni archivio dati per apprendere le funzionalità supportate in dettagli e le configurazioni corrispondenti.
