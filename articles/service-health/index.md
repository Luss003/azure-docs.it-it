# <a name="azure-service-health-documentation"></a>Documentazione di Integrità dei servizi di Azure

Integrità dei servizi di Azure è una suite di esperienze che fornisce supporto e indicazioni personalizzati quando si verificano problemi nei servizi di Azure dell'utente. Il servizio Integrità dei servizi di Azure può inviare notifiche, aiutare a comprendere gli effetti dei problemi e fornire informazioni aggiornate durante la relativa risoluzione. Può anche aiutare a preparare la manutenzione pianificata e le modifiche che potrebbero influire sulla disponibilità delle risorse.

Integrità dei servizi di Azure è composto dagli elementi seguenti:

 * **Stato di Azure**: visualizzazione globale dell'integrità dei servizi di Azure
 * **Integrità dei servizi**: visualizzazione personalizzata dell'integrità dei servizi di Azure
 * **Integrità risorse**: visualizzazione più approfondita dell'integrità delle singole risorse messe a disposizione dai servizi di Azure

Insieme, queste esperienze offrono una visualizzazione completa dello stato di integrità di Azure, al livello di granularità più idoneo per l'utente.

Per altre informazioni dettagliate su ogni servizio, vedere [Panoramica di Integrità dei servizi](overview.md)
