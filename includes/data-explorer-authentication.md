## <a name="authentication"></a>Autenticazione

Per eseguire l'esempio seguente, è necessario disporre di un'applicazione Azure Active Directory (Azure AD) e di un'entità servizio che possano accedere alle risorse. Per creare un'applicazione Azure AD gratuita e aggiungere un'assegnazione di ruolo a livello di sottoscrizione, vedere [creare un'applicazione Azure ad](/azure/active-directory/develop/howto-create-service-principal-portal). Sono necessari anche l'ID di directory (tenant), l'ID applicazione e il segreto client.
