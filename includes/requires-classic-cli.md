> [!IMPORTANT]
> Questo articolo riguarda contenuti che richiedono l'interfaccia della riga di comando classica di Azure. La versione corrente dell'interfaccia della riga di comando di Azure non supporta le funzionalità descritte in questo articolo. È quindi necessario usare l'interfaccia della riga di comando classica. 
> 
> L'interfaccia della riga di comando classica può essere installato insieme la CLI di Azure moderna, ma è consigliabile la CLI di Azure per tutti i nuovi alfabeti e le distribuzioni.
> Per installare l'interfaccia della riga di comando classica, vedere [Installare l'interfaccia della riga di comando classica di Azure](/cli/azure/install-classic-cli). 
> Per installare la versione più recente della riga di comando, vedere [installare CLI Azure](/cli/azure/install-azure-cli).
