È necessario avere un [account delle API Servizi cognitivi](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) con accesso alle API di ricerca Bing. Prima di continuare, è necessario ottenere la chiave di accesso fornita dopo aver avviato una sottoscrizione con il piano tariffario S9, come illustrato in [Prezzi di Servizi cognitivi - API Ricerca Bing](https://azure.microsoft.com/pricing/details/cognitive-services/search-api/). 

Per avviare una sottoscrizione nel portale di Azure:
1. Nella parte superiore del portale di Azure immettere 'BingSearchV7' nella casella di testo `Search resources, services, and docs`.  
2. Nella sezione Marketplace dell'elenco a discesa selezionare `Bing Search v7`.
3. Immettere `Name` per la nuova risorsa.
4. Selezionare la sottoscrizione `Pay-As-You-Go`.
5. Selezionare il piano tariffario `S9`.
6. Fare clic su `Enable` per attivare la sottoscrizione.
