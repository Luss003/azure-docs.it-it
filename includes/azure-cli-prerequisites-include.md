## <a name="prerequisite-install-the-azure-cli"></a>Prerequisiti: Installare l'interfaccia della riga di comando di Azure

Per eseguire i passaggi in questo articolo [installare Azure CLI](/cli/azure/install-azure-cli) e [accedere ad Azure](/cli/azure/authenticate-azure-cli). 

> [!NOTE]
> Se non si dispone di un account Azure, è necessario procurarsene uno. Iscriversi per una [versione di valutazione gratuita](../articles/active-directory/fundamentals/sign-up-organization.md).
> Inoltre, per seguire la procedura è necessario aver [jq](https://stedolan.github.io/jq/) o alcuni altri JSON installato una libreria o lo strumento di analisi.
